# 🚀 AI Resume Analyzer

An AI-powered Resume Analyzer that helps job seekers optimize their resumes by comparing them with job descriptions, generating ATS scores, and providing actionable feedback using advanced AI models.

Built with a modern full-stack architecture, this project demonstrates real-world software engineering practices including authentication, file processing, AI integration, state management, and cloud services.

Deployed link : https://ai-resume-analyzer-flax-chi.vercel.app/
---

## ✨ Features

### 🔐 Authentication
- Secure login & logout system  
- Protected routes  
- Persistent user sessions  

### 📄 Resume Upload
- Drag-and-drop PDF upload  
- File validation  
- Resume preview generation  
- PDF-to-image conversion  

### 🤖 AI Resume Analysis
- Resume vs Job Description matching  
- ATS (Applicant Tracking System) score generation  
- Skill gap identification  
- Content quality evaluation  
- Structure & formatting analysis  
- Tone & professionalism assessment  

---

## 📊 AI Feedback System

The AI evaluates resumes across multiple dimensions:

- Content Quality  
- Skills Match  
- Structure & Formatting  
- ATS Optimization  
- Keyword Relevance  
- Professional Tone

- Highlights missing keywords

- Improves chances of passing recruiter filters
